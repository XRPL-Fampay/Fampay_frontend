# Fampay - Family Group Wallet on XRPL
> 가족을 위한 소셜 페이먼트 월렛 – 송금은 빠르게, 현금화는 안전하게, 키 관리는 쉽게.

## 개요
해외 노동자나 유학생이 가족에게 송금할 때, 단순 송금이 아니라 가족 단위로 관리되는 소셜 페이먼트 월렛을 제공하는 서비스. 사용자가 송금하면 원장에서 자동으로 가족 구성원 지갑에 분배되고, 정기적인 생활비·학비 지급을 자동화할 수 있다. 
또한 규제 준수와 현금화 루트를 지원해, 실제 생활 속에서 바로 사용할 수 있는 가족 맞춤형 금융 인프라를 목표로 한다.

## 주요기능
### 1. 빠르고 투명한 송금 (Payment)
- 해외 노동자·유학생이 송금하면 **가족 지갑에 자동 분배**
- **Batch / Escrow** 기능으로 정기 생활비·학비 자동 지급 (이번 해커톤 필수 구현 사항)
- **Memos**로 송금 목적 기록 → 가족 간 투명한 사용 내역

## 2. 안전한 현금화 (Cash-out) → 로드맵에서 어필 + UI로 보여줌

- **PermissionedDomains** 기반, 허가받은 게이트웨이만 연결(이번 해커톤 필수 구현 사항)
데모 버전 -> 연결된 XRP 카드 지갑 주소로 전송 가능한 UI 제공 (ex. redotpay, ether.fi, kast처럼 실물카드와 연계되듯이)
- 가족은 스테이블코인을 **은행 계좌 / 모바일 머니 / ATM**에서 1:1 현금화
- PG/VAN 수수료(2~3%) 대신 **거의 제로 수준 수수료**, 정산도 수 초 내 완료

## 3. 쉬운 키 관리 (Key Management)

- **간단 UX**: 비밀번호·생체인증으로 접근, 키는 보안영역 관리
- **멀티시그 그룹 지갑**: 가족 2/3 이상 승인 시 출금 가능
- **소셜 리커버리**: 키 분실 시 가족 구성원이 함께 복구 지원
- **하이브리드 커스터디**: 사용자 선택에 따라 완전 자가보관 / 반커스터디 선택 가능
- 소셜 리커버리 → 제일 간단해보이는 방식으로 채택 후 로드맵에서 보완
- 니모닉 생성 관리 → wallet에 메서드 있음(이 방식으로 구현)
